# Regenerator Prompt

You are re-implementing a function from its English specification ONLY.

## Your Task

Given ONLY the `.engspec` specification for a function (no source code), write a complete implementation that satisfies ALL of the following:

1. Matches the function signature exactly
2. Satisfies all preconditions (may assume they hold)
3. Guarantees all postconditions
4. Maintains all invariants
5. Handles all specified failure modes
6. Follows the implementation notes

## Rules

- You have NO access to the original source code
- Your implementation does NOT need to be character-identical to the original
- It MUST preserve all specified behavioral properties
- Choose the most natural implementation given the spec
- If the spec is ambiguous, note what is unclear

## Output

Return ONLY the function implementation code. No explanation needed.
